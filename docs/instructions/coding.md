# Coding instructions

## General

Before committing any changes, you MUST build and test the entire tree, and fix any build/test breakages that occur. Code that is pushed to the repo MUST always compile, and the tests MUST pass.

All agent-generated code MUST come with 100% branch coverage. If there is any branch that cannot be tested, this exception MUST be documented in docs/tracking/coverage-exceptions.md before code coverage requirements can be lowered. Always test as many branches as possible.
